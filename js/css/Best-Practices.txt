Box Shadow 👉 Pure element (div, button, card, image) par shadow lagata hai.
Element ke box ke bahar shadow lagata hai.

Syntax
box-shadow: x-offset y-offset blur spread color;

Example
box-shadow: 0 4px 10px  5px rgba(0,0,0,0.3);
0 → left/right
4px → top/bottom
10px → blur
5px -> spread
rgba(...) → color


text-shadow: x-offset y-offset blur color; 👉 Sirf text par shadow lagata hai.
text-shadow: 2px 2px 5px gray;


box-shadow: x y blur spread color;
text-shadow: x y blur color;
✅ box-shadow me spread aur inset hota hai.
❌ text-shadow me spread aur inset nahi hota.

what we cant set inline element ? ( height, width, margin-top,bottom )

width and height: These properties are completely ignored; the element's size is strictly determined by its content.margin-top and margin-bottom: Vertical margins have absolutely no effect and will not push surrounding elements away.padding-top and padding-bottom (Layout impact): While vertical padding visually appears, it does not alter the document flow and will bleed into or overlap adjacent lines of text.transform: Properties like rotate(), scale(), or translate() do not apply to standard inline elements.

"To apply these properties while keeping the element in line with text, you must change its display property to display: inline-block or display: block."
