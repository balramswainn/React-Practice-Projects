Kab height deni chahiye?
1. Equal Height Cards
.card {
  min-height: 20rem;  //Minimum 20rem rakho, lekin content agar zyada hai toh grow kar jao.
}

Problems:-
1. But hume chahiye sabka height same ho if content kiska jyada bhi hogya toh sabka height same hi rahega esa chaiye
<div class="parent">
  <div class="card">Short text</div>
  <div class="card">Medium text</div>
  <div class="card">Very long content...</div>
  <div class="card">Short text</div>
</div>

.parent{
  display:flex;
  justify-content:center;
  align-items:center; 
  height:100vh;        
}

.card{
  width:200px;    //height nhi diya bcz we want content according  height jitna ho utna dikhe
}

o/p:-  sab content k hisab se height le liye but sab chote bade hogye 

reason :- align-items:center; Flexbox me default hota hai: align-items: stretch; Sab flex items cross-axis me equal height le lenge. but humne likha align-items:center so  Ab browser bolta hai:  Stretch mat karo.Har item ki height uske content ke hisab se rakho.Aur center me align kar do. Isliye har card alag height ka dikh raha hai.

                          align-items: stretch;  //by default hota hai ye

o/p:- sab card equal height pr hai but kuch jyada hi height le liya reason: parent pe height:100vh;   
remove that height and content ko padding de ya content cards ko decide karne de height and it will show proper
