Step 1 — Package install

Terminal:  npm install gh-pages --save-dev



Step 2 — package.json
Add: "homepage":"https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME"

Example: "homepage":"https://balram.github.io/myapp"





Step 3 — scripts add karo

"scripts": {
  "dev": "vite",
  "build": "vite build",
  "predeploy": "npm run build",
  "deploy": "gh-pages -d dist"
}




Step 4 — Vite config

vite.config.js

export default defineConfig({
  plugins:[react()],
  base:'/REPO_NAME/'
})

Example:  base:'/myapp/'

VERY IMPORTANT ⚠️



Step 5 — Deploy
npm run deploy


Step 6 — GitHub settings
Repo → Settings → Pages
Select: Branch: gh-pages
Save.


Then website live:
https://username.github.io/repo-name/
